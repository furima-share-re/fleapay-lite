## 📋 出店者側画面一覧

### 1. 出店者ダッシュボード（Next.js版）

**URL**: `/seller-dashboard?s={sellerId}`

**説明**: 出店者用のメインダッシュボード（売上サマリー、取引履歴、販売用URL・QRコード表示）

**テスト用sellerId**:
- Standardプラン: `test-seller-standard`
- Proプラン: `test-seller-pro`
- Kidsプラン: `test-seller-kids`

**動作確認URL例**:
```
http://localhost:3000/seller-dashboard?s=test-seller-pro
```

**主な機能**:
- 今日の売上サマリー（純売上、決済回数、平均客単価）
- 仕入額・利益の表示
- 最近の取引履歴
- 販売用URL・QRコード生成
- データ精度スコア表示
- プラン別の機能制限（Standardプランは一部機能制限）

---

### 2. Kidsダッシュボード（Next.js版）

**URL**: `/kids-dashboard?s={sellerId}`

**説明**: Kidsプラン専用のダッシュボード（ゲーミフィケーション要素あり）

**テスト用sellerId**: `test-seller-kids`

**動作確認URL例**:
```
http://localhost:3000/kids-dashboard?s=test-seller-kids
```

**主な機能**:
- 売上サマリー（Kids向けUI）
- バッジ・称号システム
- ミッション表示
- 統計情報（注文数、属性入力率など）

**注意**: Kidsプラン以外ではブロックされます

---

### 3. 出店者購入画面（標準版・Next.js）

**URL**: `/seller-purchase-standard?s={sellerId}`

**説明**: 出店者用のレジ画面（カメラ撮影、AI解析、QRコード決済）

**テスト用sellerId**: `test-seller-pro` または `test-seller-kids`（Pro/Kidsプラン必須）

**動作確認URL例**:
```
http://localhost:3000/seller-purchase-standard?s=test-seller-pro
```

**主な機能**:
- カメラ撮影による商品認識
- AI解析による自動集計
- 手動入力フォーム
- QRコード決済リンク生成
- 現金/キャッシュレス選択

**注意**: Standardプランではブロックされます

---

### 4. 出店者登録ページ

**動作確認URL例**:

## 🔧 動作確認手順

### 1. ローカル環境での確認

```powershell
# 開発サーバー起動（既に起動している場合は不要）
npm run dev

# ブラウザで以下を開く
# http://localhost:3000/seller-dashboard?s=test-seller-pro
```

### 2. 各画面の確認ポイント

#### 出店者ダッシュボード
- [ ] 売上サマリーが正しく表示される
- [ ] 取引履歴が表示される
- [ ] QRコードが生成される
- [ ] プラン別の機能制限が正しく動作する
- [ ] リアルタイム更新が動作する（30秒ごと）

#### Kidsダッシュボード
- [ ] Kidsプランでアクセス可能
- [ ] バッジ・称号が表示される
- [ ] ミッションが表示される
- [ ] Standard/Proプランではブロックされる

#### 出店者購入画面
- [ ] カメラ機能が動作する
- [ ] AI解析が実行される
- [ ] QRコードが生成される
- [ ] Standardプランではブロックされる

## 📊 テスト用データ

### テスト用sellerId一覧

| sellerId | プラン | 説明 |
|----------|--------|------|
| `test-seller-standard` | Standard | Standardプラン（一部機能制限） |
| `test-seller-pro` | Pro | Proプラン（全機能利用可能） |

### APIエンドポイント

出店者関連のAPI:
- `/api/seller/summary?s={sellerId}` - 売上サマリー取得
- `/api/seller/kids-summary?s={sellerId}` - Kidsサマリー取得
- `/api/seller/analytics?s={sellerId}&period={period}` - 売上分析
- `/api/seller/check-id?id={id}` - 出店者ID確認

---

## ⚠️ 注意事項

1. **プラン制限**: 各画面はプランによってアクセス制限があります
   - Standardプラン: 出店者購入画面は利用不可
   - Kidsプラン: Kidsダッシュボードのみ利用可能
   - Proプラン: 全機能利用可能

2. **認証**: 現状、sellerIdパラメータのみでアクセス可能（認証なし）

3. **データ**: テスト用sellerIdには実際の取引データが存在しない場合があります
